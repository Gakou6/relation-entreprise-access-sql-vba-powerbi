# Scripts VBA

Ce dossier contient les procédures VBA utilisées dans Access pour automatiser la création, l’alimentation et la réinitialisation des tables.

Les scripts VBA permettent notamment :
- d’importer les fichiers Excel sources ;
- d’alimenter les tables définitives ;
- de supprimer les doublons ;
- de respecter l’ordre d’insertion lié aux clés étrangères ;
- de faciliter les tests de la base.
